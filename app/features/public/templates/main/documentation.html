CalltoConvey Documentation

Everything you need to set up, run, and optimise your outbound communication campaigns.

Quick Start Guide

Get your first campaign running in under 10 minutes. Follow these steps:

1

Register your Organisation

Go to Get Started and complete the 3-step organisation registration form. Your account will be reviewed and activated by a platform administrator — usually within a few hours.

2

Configure Communication Keys

Navigate to Organisation → Settings → Integrations and enter your Twilio Account SID, Auth Token, and phone number. Alternatively, use platform-default credits if provided.

3

Create a Module

Go to Modules and click New Module. Define your custom fields (name, phone, amount, etc.) then upload your audience data via Excel or CSV.

4

Write a Script

Navigate to Scripts and create a message template. Use {{field_name}} placeholders to personalise the message for each recipient.

5

Launch a Campaign

Go to Campaigns → New Campaign, select your module, choose the channel (Voice or WhatsApp), assign workers, pick your script, and click Launch.

Modules & Data

Modules are the core data containers in CalltoConvey. Each module represents a specific communication use-case (e.g., "Student Fee Reminders", "Loan EMI Follow-ups").

Creating Fields

Each module can have unlimited custom fields of the following types:

  • Text — Free-form text like names, addresses, or IDs
  • Number — Numeric values like fees, amounts, or scores
  • Boolean — True/false flags like "payment_done" or "call_answered"
  • Calculated — Auto-computed values based on other field values using expressions

Importing Data

Upload audience data as .xlsx or .csv. CalltoConvey will automatically map your column headers to your module fields. You can also export the full dataset at any time.

Tip: Ensure your spreadsheet has a phone column with valid 10-digit mobile numbers. This is required for both voice and WhatsApp campaigns.

Automated Voice Calls

CalltoConvey uses Twilio's voice API to auto-dial each recipient. The system plays a custom audio script generated from your text template.

Text-to-Speech Engines

  • Google TTS (gTTS) — Supports 40+ languages, natural-sounding voice
  • Microsoft Edge TTS — Neural voices, highly natural, supports Indian English, Hindi, Tamil, Telugu, and more

Call Status Tracking

Twilio sends real-time webhook callbacks to CalltoConvey. Every call result is logged instantly: completed, busy, no-answer, failed.

API Key Configuration

Configure your Twilio credentials from Organisation → Settings → Integrations.

TWILIO_ACCOUNT_SID = "ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
TWILIO_AUTH_TOKEN = "your_auth_token_here"
TWILIO_PHONE_NUMBER = "+91XXXXXXXXXX"

CalltoConvey encrypts your API credentials using AES-256 before storing them. They are never exposed in logs or API responses.

Security & MFA

CalltoConvey includes enterprise-grade security out of the box:

  • Multi-Factor Authentication (MFA) — TOTP-based 2FA via Google Authenticator or Authy
  • Brute-Force Protection — Accounts are locked after repeated failed login attempts
  • Session Management — Secure, fingerprinted sessions with CSRF protection
  • Multi-Tenant Isolation — Zero data crossover between organisations
  • Content Security Policy — XSS and clickjacking protection headers on all pages
