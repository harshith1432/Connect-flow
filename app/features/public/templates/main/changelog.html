Every release, every improvement, every fix — documented here so you always know what's new in ConnectFlow.

May 2026

v2.5.0
Major Release — Billing, Invoicing & Premium Dashboards LATEST
New
  • Real-time Payments Ledger — live payment history per organisation
  • Commercial Tax Invoice generation with CGST/SGST breakdown and digital seal
  • Enterprise Checkout page with Razorpay integration support
  • Subscription Plans page with usage tracking and visual utilisation bars
  • Premium dual-pane Profile layout for both Organisation Admin and Worker dashboards
  • Dark Mode / Light Mode theme switcher across all dashboard views
  • Dedicated Billing sidebar with Payments, Invoices, Plans, and Subscriptions navigation
  • Centralised logging system (app/core/logging_system.py) with rotating file handler
Improved
  • Notification detail modal — clicking a notification now shows full details
  • Organisation Profile page now uses split-panel premium layout
  • Worker Profile page redesigned to match Organisation Admin profile consistency
  • Sidebar no longer shows email address under user profile — cleaner design
Fixed
  • Fixed BuildError for org.payments endpoint — now correctly resolves to org.process_payment
  • Fixed UndefinedError for limits variable on the Plans page
  • Fixed wrong sidebar rendering on Organisation Profile (was showing base sidebar instead of billing sidebar)

Apr 2026

v2.0.0
Major Release — Campaign Engine & Worker Roles
New
  • Full campaign management: create, assign, launch, and track campaigns
  • Worker role with dedicated workspace — campaigns, scripts, modules, groups
  • Real-time delivery log tracking via Twilio webhooks
  • Automated voice call dispatch with Microsoft Edge TTS and Google TTS
