Module Fields Guide
1. How to create Field Sections

Field sections allow you to map out exactly what data you want to collect. You define the Attribute Name (e.g. "Customer Name") and the Data Logic Type.

Available Types
  • Standard Text: Regular text input
  • Number Value: Strictly numerical data
  • Phone / Mobile: Connects with WhatsApp automatically if "Use for Communication" is toggled
  • Email Address: Validates standard email formats
  • Date Picker: Calendar selection
  • Dropdown Selection: Enter comma-separated options (e.g. Paid, Pending, Failed)
2. How to use Calculated Formulas

Calculated formulas allow you to perform automatic mathematical operations on other fields in the same record.

Note: You must reference other fields exactly by their name, enclosed in curly brackets {}.

Example Scenario: You want to calculate a final invoice total based on "Hours Worked" and "Hourly Rate", plus a $50 fee.

({Hours Worked} * {Hourly Rate}) + 50
Supported operators: +, -, *, /, ( )
